# 🎯 Purpose

Create an LLM-friendly ritual that reviews PRDs from different persona perspectives (e.g. Product Owner, Systems Engineer, Architect).

# 🔧 Prompt

> build a CLI or scripted tool that takes a PRD
> simulates feedback from different product personas
> returns critiques, gaps, and confidence ratings
> optionally logs feedback into the PRD file under `## Feedback`
