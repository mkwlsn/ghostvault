Enable rapid capture of ideas and chat notes into fully valid PRD markdown files.

> take a plaintext list of stories, tasks, and rationale
> output a valid semantic markdown PRD with:
> - frontmatter autogenerated
> - correct story formatting + sections
> - YAML blocks with default metadata
> include a short summary block and output filename
